Kraken for WordPress

An add-on to the Kraken boilerplate for WordPress developers.

Download on GitHub

Or keep reading to learn more...

Essential Templates

Kraken for WordPress comes with the essential templates you need to start building a WordPress site:

  • Individual pages and posts.
  • A full list of your blog posts.
  • A comments template.
  • Posts by category, tag, author, etc.
  • Search results.
  • The header, navigation, and footer.
  • 404 errors.
  • Placeholders for icons and images.

Each template is preloaded with useful functions for things like titles, post dates, page content and more. They also include all of the appropriate HTML5 semantic elements.

Useful Functions

The functions.php file features a few useful functions to help improve site performance and make development easier:

  • Load externally hosted JavaScript files.
  • Replace the built-in version of jQuery with a CDN-hosted version.
  • A simple search form function and shortcode.
  • Remove the unneccessary junk WordPress adds to the header.
  • Exclude pages from search results.
  • Remove trackbacks from comments.
  • Minify your HTML.

WordPress Styling

The style.css file includes a WordPress theme header, and a few WordPress-specific classes that control image and avatar styling.

Built for Kraken

Kraken for WordPress was built as an add-on for the Kraken boilerplate.

Aside from the theme header and a few WordPress-specific classes, the stylesheet is empty. Drop in the Kraken boilerplate, or write your own stylesheet from scratch.
