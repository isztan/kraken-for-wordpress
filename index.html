Essential Templates

Kraken for WordPress comes with the essential templates you need to start building a WordPress site:

  • Individual pages and posts.
  • A full list of your blog posts.
  • A comments template.
  • Posts by category, tag, author, etc.
  • Search results.
  • The header, navigation, and footer.
  • 404 errors.
  • Placeholders for icons and images.

Each template is preloaded with useful functions for things like titles, post dates, page content and more. They also include all of the appropriate HTML5 semantic elements.

Useful Functions

The functions.php file features a few useful functions to help improve site performance and make development easier:

  • Load external JavaScript files.
  • Replace the built-in version of jQuery with a CDN-hosted version.
  • A simple search form function and shortcode.
  • Remove the unneccessary junk WordPress adds to the header.
  • Exclude pages from search results.
  • Remove trackbacks from comments.
  • Prevent internal links from showing up as trackbacks.
  • Minify your HTML.

How to Use Kraken for WordPress

Kraken for WordPress assumes a working knowledge of HTML and CSS. A little familiarity with PHP is useful but not required.

1. Add your styles.

The style.css file includes a WordPress theme header, and a few WordPress-specific classes that control image and avatar styling. The stylesheet is otherwise empty.

Drop in the Kraken boilerplate, or write your own stylesheet from scratch.

2. Structure your HTML.

The PHP templates include commonly used functions and content, but not much style. Structure your HTML as desired.

3. Pat yourself on the back.

Seriously, you're done. Nice work! Modify as needed.
